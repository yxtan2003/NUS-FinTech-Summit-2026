Step 1: git pull origin main into local repository
Step 2: Open new Terminal
Step 3: Navigate to the local repository
Step 4: To simulate minting of NFT: navigate using "cd minting"
Step 5: npm install dotenv express firebase firebase-admin xrpl
Step 6: run "node simulateMINT.js" in the Terminal

If done correctly, a message containing:
"NFT minted: {
  firestore_id: '...',
  xrpl_token_id: '...'
}"
should be returned.



How to verify if batch of drugs is authentic:

Step 7: run node app.js
Step 8: Open up cmd and run:
curl -X POST http://localhost:3000/api/verify -H "Content-Type: application/json" -d "{\"chipId\":\"YOUR_CHIP_ID_HERE\"}"

If done correctly, a message containing:
{
  chipId: 'CHIP_ID',
  result: 'AUTHENTIC/COUNTERFEIT',
  timestamp: '...'
}
should be returned
